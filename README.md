# Wapate (WP)

> A global creative network — discover people, events, and communities.
> *No creative should feel alone.*

Formerly developed as WeKonnect.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
